# Jet Engine Predictive Maintenance: RUL Prediction

This project focuses on predicting the **Remaining Useful Life (RUL)** of aircraft jet engines using the **NASA CMAPSS dataset**. By analyzing sensor data, we aim to provide an early warning system to optimize maintenance schedules and reduce operational costs.

## 🚀 Highlights
- **Model:** Random Forest Regressor
- **Target Accuracy:** ~11.92 MAE (Mean Absolute Error)
- **R2 Score:** 81.37%
- **Key Techniques:** RUL Clipping, Rolling Averages, Feature Scaling, Feature Importance Analysis.

## 📊 Dataset
The dataset contains simulated degradation data from turbofan engines. 
- 21 sensor readings (temperature, pressure, speed, etc.)
- Operational settings
- Time in cycles
